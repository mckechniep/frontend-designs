    What is this component library?

    A zero-build vanilla HTML/CSS/JS starter with 8 style profiles, light/dark theming, and accessible components you can drop into any project.

    Do I need a build tool?

    No. Everything works with plain HTML, CSS, and JavaScript. Just open index.html in a browser or use any static file server.

    How do the style profiles work?

    Each profile is a CSS file that overrides design tokens (colors, fonts, radii, shadows). Switching profiles swaps the linked stylesheet, and all components update instantly.

    Can I add my own profile?

    Absolutely. Copy any existing profile CSS file, adjust the token values, add an entry to profiles.json, and your new profile appears in the switcher.
