# Dashboard A/B Prompt

Use this exact base prompt text for both runs:

`Build an operations dashboard with KPI cards, a responsive trend chart, and a sortable/filterable transactions table. Include clear loading/empty/error states and keyboard-accessible controls.`

Run variants:

- With skill: prepend `Use $kickass-frontend.`
- Without skill: do not mention the skill.

Font delivery lock (must be identical for both runs):

- `A)` skill-shipped fonts (Recommended)
- `B)` skill fonts via Google-hosted web fonts
- `C)` local/system (final safety fallback)
- `D)` other custom fonts (user-provided)
