# Landing Page A/B Prompt

Use this exact base prompt text for both runs:

`Build a modern landing page for a B2B analytics product. Include hero, social proof, feature grid, pricing preview, and final CTA. Must be responsive, accessible, and visually distinctive on desktop and mobile.`

Run variants:

- With skill: prepend `Use $kickass-frontend.`
- Without skill: do not mention the skill.

Font delivery lock (must be identical for both runs):

- `A)` skill-shipped fonts (Recommended)
- `B)` skill fonts via Google-hosted web fonts
- `C)` local/system (final safety fallback)
- `D)` other custom fonts (user-provided)

Save final screenshots as:

- `with-skill-desktop.png`
- `with-skill-mobile.png`
- `without-skill-desktop.png`
- `without-skill-mobile.png`
