# Golden Example: Dashboard / Retro Terminal

## Scope Lock

- output: `react`
- styling: `tailwind`
- profile: `retro-terminal`
- theme: `dark`

## Signature

- command-center dashboard with CRT green telemetry rows and hard-edged shell modules
- live alert feed + process table + terminal stat clusters
- mono-first typography, phosphor glow, and strict console spacing rhythm
- keep it operational and credible; avoid rounded cards or soft product-app affordances
