# Golden Example: App Screen / Cyberpunk Neon

## Scope Lock

- output: `nextjs`
- styling: `css`
- profile: `cyberpunk-neon`
- theme: `dark`

## Signature

- operator-console workspace with clipped surfaces, neon edge treatment, and tactical labels
- command area + scenario controls + live system rail or event stream
- display type stays futuristic while dense controls remain mono/terminal-led
- effects stay scoped and legible; do not turn the screen into unreadable glow soup
