# Golden Example: Dashboard / Sunset Gradient

## Scope Lock

- output: `react`
- styling: `tailwind`
- profile: `sunset-gradient`
- theme: `dark`

## Signature

- cinematic enterprise dashboard with warm horizon glow and glass-framed data modules
- stat ribbon + comparative metrics + roadmap or launch-status board
- serif display meets clean sans UI with sunset poster depth carried into product surfaces
- keep the pacing elegant and presentational, not flat and dashboard-generic
