# Golden Example: App Screen / Modern SaaS

## Scope Lock

- output: `nextjs`
- styling: `css`
- profile: `modern-saas`
- theme: `dark`

## Signature

- premium workspace or settings screen with serif-led section titles and clean product controls
- split main canvas with primary form/action column and trust-building summary rail
- warm neutral surfaces, restrained glass, and amber emphasis around the primary action path
- polished B2B product feel; never drift into generic admin tooling
