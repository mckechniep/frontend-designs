# Golden Example: Dashboard / Modern SaaS

## Scope Lock

- output: `react`
- styling: `tailwind`
- profile: `modern-saas`
- theme: `dark`

## Signature

- premium product-ops dashboard with warm slate field and serif-led heading moments
- metric rail + revenue/retention snapshot + plan conversion or pipeline board
- selective glass and amber/sage accent use; avoid generic flat SaaS card repetition
- proof-oriented dashboard composition that still reads like the same product family as the landing surface
