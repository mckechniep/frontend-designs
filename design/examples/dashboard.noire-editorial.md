# Golden Example: Dashboard / Noire Editorial

## Scope Lock

- output: `react`
- styling: `tailwind`
- profile: `noire-editorial`
- theme: `dark`

## Signature

- dossier-style executive dashboard with editorial hierarchy instead of commodity BI layout
- narrative KPI strip + case activity feed + evidence or records table
- serif contrast and classified labels should guide attention without hurting scannability
- preserve near-black luxury tone; avoid beige magazine drift or generic startup metrics cards
