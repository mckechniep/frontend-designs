# Golden Example: App Screen / Sunset Gradient

## Scope Lock

- output: `nextjs`
- styling: `css`
- profile: `sunset-gradient`
- theme: `dark`

## Signature

- founder-facing workspace with warm glass cards, milestone tracking, and launch-readiness framing
- split screen with main action canvas and luminous progress or summary rail
- cinematic gradients should support focus, not overpower controls
- premium launch-platform feeling rather than conventional settings page minimalism
