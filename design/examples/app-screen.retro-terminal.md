# Golden Example: App Screen / Retro Terminal

## Scope Lock

- output: `nextjs`
- styling: `css`
- profile: `retro-terminal`
- theme: `dark`

## Signature

- terminal-native app workspace with stacked forms, toggles, and logs inside a shell-like frame
- strong monochrome command posture with occasional amber/red status accents
- zero-radius geometry and CRT overlays remain part of the identity
- dense but readable operational controls; never polish it into glossy SaaS
